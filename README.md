# picture_gallery_es6

Prerequisites
=============
* Node / NPM


Initial Set Up
==============
* run `npm install -g webpack`
* git clone `https://github.com/yeongk/picture_gallery_es6.git`


Build
=====
* run `npm install` at the root folder
* run `webpack`
* to watch, run `webpack --watch`
* to build uglified js, run `ODE_ENV=production webpack`


Run
=====
* click on `index.html`
